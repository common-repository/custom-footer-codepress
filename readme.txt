== Description ==

Are you ready to increase your productivity? The Custom Footer CODEPRESS WordPress plugin will allow you to quickly and conveniently add simple footer in your pages, posts and all your site.

Custom Footer CODEPRESS - does not collect any information outside your WordPress installation.   


== Features ==

* Custom Footer Codepress - Just activate the Custom Footer CODEPRESS plugin and the footer will appear on your page, showing the description of your Wordpress installation and also the current year.

== Installation ==

Use the Add New Plugin feature within your WordPress site and search for "Custom Footer - CODEPRESS" to install and activate it. You may also download the archive and:

1. Upload 'codepress-custom-footer' to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu within WordPress
3. Go to Settings -> Custom Footer - CODEPRESS and customize the plugins options
